---

## First-week setup

1. Login (phone OTP recommended)  
2. **Team** — add another trustee + counter clerk  
3. **Trust / 80G** — legal name, registration, 80G no., trust PAN (from CA)  
4. **Money desk** — practice ₹1–101 entry + print  
5. Share donate link on WhatsApp status / site  
6. Open **Weekly report** once  

---

## Daily controls you should have

| Control | Page | Who |
|---------|------|-----|
| Issue counter receipts | Money desk | Clerk + you |
| See live totals | Dashboard | You |
| Approve team access | Team | You |
| Change plan | Billing | You |
| Export for CA | Trust → 10BD (Pro) | You + CA |

---

## What you should not expect MandirOS to do

- Replace full Tally accounting  
- File income tax for you  
- Provide legal tax advice  
- Physical POS hardware (yet)  

---

## Security for boards

- Only trustees/admins get dashboard  
- Remove staff who leave the same day  
- Don’t share login OTPs on public groups  
- API keys (Pro) only with IT
